# Nexo AI - Landing Page

Esta é a Landing Page do Nexo AI, criada com Nuxt 3 e Tailwind CSS.

## Como rodar

1. Instale as dependências:
```bash
npm install
# ou
pnpm install
```

2. Rode o servidor de desenvolvimento:
```bash
npm run dev
# ou
pnpm run dev
```

Acesse http://localhost:3000

## Estrutura

- `app.vue`: Contém todo o código da Landing Page (Single Page)
- `nuxt.config.ts`: Configuração do Nuxt e Tailwind
- `tailwind.config.js`: Configuração de cores e fontes
